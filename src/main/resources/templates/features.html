EventMaster

A web application for creating, managing, and explorinig different types of events.

You will able to:

Create new events and manage them
Add vendors and control budget
Add guests manually or share with them the invitation code
Create event tasks and assign them to assistant users
See publicly available events and join them
In case of private events, join them by invitation code
And many more...
